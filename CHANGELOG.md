# Changelog

## v1.0.1 — 2026-07-19

- แปล loading tips ครบ 100 ประโยค รวมข้อความที่เกมดึงตรงจากฐานข้อมูล `sv-SE`
- รองรับ loading tips ที่ถูกแปลงเป็นตัวพิมพ์ใหญ่หรือมีช่องว่างส่วนเกิน
- แปลเมนู บทช่วยสอน แดชบอร์ด อีเมล และบทสนทนาครบตาม inventory ที่สกัดได้
- เพิ่ม TextMeshPro fallback สำหรับภาษาไทย และแปลงชื่อวัน/เดือนจากสวีเดนเป็นไทย
- ตรวจ placeholder และโครงสร้างคำแปลโดยไม่พบ hard failure
