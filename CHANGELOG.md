# Changelog

## v1.0.2 — 2026-07-19

- แก้อาการเกมหน่วงหรือค้างเมื่อเดินวัน กด Continue และส่งข้อเสนอสัญญา
- ปรับระบบแปลข้อความ runtime ให้กรอง template ล่วงหน้า จำกัดเวลา regex และ cache ผลลัพธ์
- เพิ่ม `Uninstall-ThaiMod.cmd` สำหรับถอนมอดด้วยการดับเบิลคลิก
- ตัวถอนติดตั้งเก็บ MelonLoader ไว้อัตโนมัติเมื่อพบมอดหรือปลั๊กอินอื่น

## v1.0.1 — 2026-07-19

- แก้ตัวเรียกติดตั้งบน Windows ให้ใช้รูปแบบไฟล์ ASCII/CRLF และ PowerShell UTF-8 BOM ป้องกันคำสั่งแตกหรือภาษาไทยเพี้ยน
- เพิ่มแพ็ก Full Installer ที่รวม MelonLoader v0.7.3 x64 จากทางการ
- เพิ่มตัวติดตั้งแบบดับเบิลคลิก ค้นหา Steam Library และตรวจ checksum อัตโนมัติ
- แปล loading tips ครบ 100 ประโยค รวมข้อความที่เกมดึงตรงจากฐานข้อมูล `sv-SE`
- รองรับ loading tips ที่ถูกแปลงเป็นตัวพิมพ์ใหญ่หรือมีช่องว่างส่วนเกิน
- แปลเมนู บทช่วยสอน แดชบอร์ด อีเมล และบทสนทนาครบตาม inventory ที่สกัดได้
- เพิ่ม TextMeshPro fallback สำหรับภาษาไทย และแปลงชื่อวัน/เดือนจากสวีเดนเป็นไทย
- ตรวจ placeholder และโครงสร้างคำแปลโดยไม่พบ hard failure
